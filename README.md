# Go Code Examples

This repo contains code samples from my talk - "Rapid API Development in Go". Each example is numbered, and each example can be built an run.

## Build and Run

To build and run an example:

1. `cd ./<example-dir>`
2. `go build`
3. `./<binary-name>`
